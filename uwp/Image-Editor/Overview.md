# UWP Image Editor (SfImageEditor) Overview

The SfImageEditor is a very handy tool used to edit an image by annotating with free-hand drawing paths, text, and built-in shapes. It also allows you to crop, rotate, and flip the image. The SfImageEditor control has a built-in toolbar that helps you to perform the editing operations.

## Key features

* Crop
* Rotate
* Flip
* Support to add shapes such as Rectangle, Circle, Arrow, Text and Path
* Undo and Redo support
* Zoom and pan operations
* Save the image to the device
* Built-in toolbar
