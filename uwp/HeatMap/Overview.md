# Overview of UWP HeatMap (SfHeatMap)

**Essential® HeatMap UWP** represents tabular data values as gradient colors instead of numbers. Low and high values are different colors with different gradients.

![Overview_Image](Images/Overview_img1.jpeg)

**Key features:**

* 2 types of data source mapping (TableMapping, CellMapping)
* Color mapping
* Legend
* Virtualization
