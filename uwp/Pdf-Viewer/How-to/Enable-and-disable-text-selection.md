# Enable and disable text selection in UWP PDF Viewer
The text selection feature of the PDF viewer can be disabled as shown in the following code.
{% tabs %}
{% highlight c# %}
pdfViewer.IsTextSelectionEnabled = false;
{% endhighlight %}
{% highlight vbnet %}
pdfViewer.IsTextSelectionEnabled = False
{% endhighlight %}
{% endtabs %}
