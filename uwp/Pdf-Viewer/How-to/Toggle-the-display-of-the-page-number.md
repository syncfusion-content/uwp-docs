## Toggle the display of the page number
The following code example can be used to disable the display of the page number on the top left corner of each page. 
{% tabs %}
{% highlight c# %}
pdfViewer.ShowPageNumber = false;
{% endhighlight %}
{% highlight vbnet %}
pdfViewer.ShowPageNumber = False
{% endhighlight %}
{% endtabs %}
