# UWP Kanban Board (SfKanban) Overview

The Kanban control is an efficient way to visualize a workflow at each stage of completion. Kanban helps to define elegant planning and clear visualization of work progression. SfKanban also provides many features that are used to monitor the progressing tasks in the software development cycle. 

![Overview of SfKanban in UWP](SfKanban_images/SfKanban_img1.png)


## Key features

* Workflows – It allows the user to control the transitions between tasks from one category (status) to another.
* Customization – Kanban cards and placeholders can be customized. 
* WIP Limit – Visualize the Work-in-Progress limit.
* Indicator color customization – Indicator color for Kanban cards can be customized.
* Dragging events – Dragging events occur while rearranging or repositioning Kanban cards.
